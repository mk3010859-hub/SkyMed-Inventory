# ✈️ SpiceJet SkyMed Inventory System

## 🚀 Live Demo
https://mk3010859-hub.github.io/SkyMed-Pro/

## 📋 Features
- ✅ Complete Medical Inventory Management
- ✅ Stock Inbound/Outbound with Excel Import
- ✅ Real-time Dashboard with Analytics
- ✅ Expiry Alerts & Low Stock Warnings
- ✅ Supabase Cloud Sync
- ✅ Excel Reports & Backup

## 🔧 Tech Stack
- HTML + CSS + JavaScript
- Supabase (PostgreSQL)
- SheetJS (Excel Export/Import)
- GitHub Pages

## ☁️ Supabase Setup
1. Create tables: medicines, inbound, outbound, deleted_items, config
2. Update API keys in `index.html`
3. Enable RLS policies (optional)
